# MKT-KVH – Internal Task Management

Web quản lý công việc nội bộ cho team Digital Marketing (4 người):
- Leader
- Content
- Quay dựng
- Thiết kế

## Mục tiêu
- Quản lý task theo dạng Kanban
- Phân công công việc theo role
- Theo dõi deadline, trạng thái và comment
- Triển khai miễn phí trên Vercel

## Tech Stack
- Next.js (App Router)
- React
- CSS thuần
- Vercel (Deploy)

## Bắt đầu nhanh
```bash
npm install
npm run dev
```

Truy cập `http://localhost:3000` để xem giao diện quản lý công việc.

## Deploy Vercel
1. Push repo lên GitHub.
2. Import project vào https://vercel.com.
3. Deploy với gói miễn phí.
